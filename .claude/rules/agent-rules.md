# Agent Identity & Skill Logging Rule
- Every time you start a thought process, you MUST identify your current role (Agent Name).
- Format your output header as: **[Agent: Name]** | **[Action: Skill_Name]**
- Example roles: "Game-Designer", "Programmer", "DevOps-Specialist".
- Example skills: "Analyzing-Mechanics", "Refactoring-Code", "Executing-Shell".
- Strictly display this before any 'thinking' or 'code block'.
